{% filter markdown %} Here are some handpicked open source projects I've worked on in the past. For a full list of my contributions, see https://github.com/daboross. --- ### qx.lc An open source pastebin and short-link server. This was mostly a learning project for me, but it is also continually run at http://qx.lc, so check it out! The server was built in Python, using the Flask web framework. - May 2014 - October 2014 - [GitHub](https://github.com/daboross/qxlc) --- ### fern-rs A simple logging backend module for the [`rust`](https://www.rust-lang.org/) language. This allows sending logging messages to multiple log backends as well as filtering each backend based off of message priority. - December 2014 - May 2015 - [GitHub](https://github.com/daboross/fern-rs) --- ### SkyWars A server-side Minecraft Minigame where players fight to the death on floating islands. I built it when I was much more part of the Minecraft community, and I've continued add small updates and maintain it since. It is my most used open source Bukkit plugin, with over 400,000 downloads to date. - July 2013 - July 2016 - [Documentation](https://dabo.guru/projects/skywars/) - [Statistics](https://dabo.guru/statistics/skywars/) - [GitHub](https://github.com/SkyWars/SkyWars) --- ### `mc-cli` `mc-cli` is a command line client for Minecraft. It uses an [open source library](https://github.com/Steveice10/MCProtocolLib) for connecting to Minecraft servers and [JLine](https://github.com/jline) for making a nice interface. It allows for 10+ fake clients to connect to the same running server, and is useful for testing plugin features which require many players to be online to function. - September 2013 - September 2014 - [GitHub](https://github.com/daboross/minecraft-commandline-interface) --- ### daboross.net The source code for this website, and all interactive functionality. - [Github](https://github.com/daboross/dabo.guru) {% endfilter %} {% endblock %}
